== Description ==

Plugin permettant de mettre en place plusieurs exigences imposées par la loi Hamon 2014 dans une boutique woocommerce par Devictio
Cette extension contient :

- Un Widget permettant d'afficher dans une sidebar les informations obligatoires à propos du droit de rétractation et des moyens de paiement
- Un shortcode [woo_loi_hamon_formulaire], permettant d'afficher le formulaire de demande de rétractation
- Un shortcode [woo_loi_hamon_adresse], permettant d'afficher l'adresse postale où envoyer la demande de rétractation
- Création d'une page contenant un texte d'explication à propos du droit de rétractation et contenant le formulaire de rétractation
- Ajout des CGV à la fin de chaque email envoyé
- Ajout de la mentions "avec obligation de paiement" pour chaque méthode de paiement
- Ajout d'un bloc "délai de livraison" sur chaque produit (admin et front)
- Envoi d'un email de confirmation de demande de rétractation
- Vérification de la présence d'une page de CGV (Cela permet d'avoir la validation des CGV à chaque commande)

== Installation ==

This section describes how to install the plugin and get it working.
